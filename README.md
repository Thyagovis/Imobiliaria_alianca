# Landing Page – Imobiliária

Landing page simples e responsiva para uma imobiliária fictícia, desenvolvida utilizando apenas HTML e CSS. O projeto tem como objetivo apresentar a empresa, destacar imóveis disponíveis e facilitar o contato com possíveis clientes.

## Estrutura da página

- Apresentação (Hero)  
  Frase institucional com foco em branding:  
  “Unindo você ao seu sonho”

- Melhores imóveis disponíveis  
  Exibição de imóveis em destaque com imagens e valores

- Nossos serviços  
  Descrição dos principais serviços oferecidos pela imobiliária

- Contato  
  Seção destinada ao contato com a empresa

## Tecnologias utilizadas

- HTML5  
- CSS3
