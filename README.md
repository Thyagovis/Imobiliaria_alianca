# Imobiliária Aliança

Landing page institucional desenvolvida para uma imobiliária fictícia, com o objetivo de apresentar a empresa, destacar imóveis disponíveis e facilitar o contato com clientes interessados.

## Sobre o projeto

A **Imobiliária Aliança** tem como propósito conectar pessoas ao imóvel ideal, oferecendo soluções confiáveis e atendimento personalizado.

Este projeto consiste em uma landing page responsiva, focada em clareza, organização das informações e boa experiência do usuário.

## Estrutura da página

A landing page é composta pelas seguintes seções:

### 1. Apresentação (Hero)

Seção inicial com uma mensagem de destaque da marca e um breve texto institucional, reforçando o slogan:

**“Unindo você ao seu sonho.”**

### 2. Melhores imóveis disponíveis

Área destinada à exibição de imóveis em destaque, contendo:
- Imagens ilustrativas
- Breve descrição
- Valores

### 3. Nossos serviços

Seção que apresenta os principais serviços oferecidos pela imobiliária, como:
- Compra e venda de imóveis
- Consultoria de investimento
- Gestão de propriedades
- Regularização

### 4. Contato

Espaço dedicado para que o usuário entre em contato com a imobiliária, contendo formulário de contato com:
- Nome completo
- E-mail
- Telefone
- Mensagem

## Tecnologias utilizadas

- HTML5
- CSS3

## Objetivo

Este projeto tem fins educacionais e demonstrativos, sendo ideal para:
- Portfólio pessoal
- Estudos de front-end
- Base para projetos reais de landing pages
